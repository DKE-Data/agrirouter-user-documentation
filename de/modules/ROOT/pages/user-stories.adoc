= agrirouter User Stories

In diesem Abschnitt sind die verschiedenen Einsatzgebiete des agrirouters dargestellt – von der Aussaat über die Düngung bis zur Ernte. Die hier gezeigten User Stories sind kurze Praxisbeispiele: Sie zeigen, wie Landwirte, Lohnunternehmer, Berater, Hersteller und andere Beteiligte den agrirouter in ihrem Arbeitsalltag nutzen – zum Beispiel um Daten einfacher auszutauschen, Arbeitsabläufe zu vereinfachen oder Maschinen unterschiedlicher Hersteller besser zu vernetzen.

Über die nachfolgenden Verweise lässt sich gezielt in die einzelnen Themenbereiche einsteigen.
